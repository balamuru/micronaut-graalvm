
## Summary
This is a micronaut/GraalVM showcase application

## Instructions
### Creation
```
mn create-app example.micronaut.micronaut-graalvm --features=graalvm
```

## Build
### Gradle Wrapper Build
```
./gradlew build
```
### Docker Container Build
```
./docker-build.sh
```
